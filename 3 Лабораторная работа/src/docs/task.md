ВАРИАНТ 10

Предусмотреть формирование элементов целочисленного массива двумя вариантами:
- ввод чисел с клавиатуры,
- с помощью функции генерации случайных чисел.

Выбор типа ввода данных должен осуществляться уже после запуска программы.

3. Упорядочить элементы массива по возрастанию.

Реализовать программу в двух вариантах: в первом — при обращении к элементам массива использовать индексы, во втором — через операцию разыменования.  
Напечатать адрес массива и адрес всех его элементов.  
Сделать структурную схему алгоритма.
