# **Лабораторная работа**

## **Алгоритмы сортировки данных**

### **Тема:**

Исследование и сравнение алгоритмов сортировки:

* сортировка обменом («пузырьковая»),
* сортировка выбором,
* сортировка вставкой.

---

# **ЗАДАНИЕ**

### **1. Ввод данных**

Размерность массива **n** вводится с клавиатуры.
Выделяется память под динамический массив из **n элементов**.

Значения элементов массива генерируются с помощью функции `rand()`.

---

# **2. Требуется выполнить сортировки для каждого алгоритма:**

Для исходного массива из **n элементов** выполнить сортировку для:

1. **Случайной последовательности чисел** (массив заполняется случайными значениями).
2. **Уже отсортированных последовательностей**:

   * в **возрастающем** порядке (лучший случай для некоторых алгоритмов),
   * в **убывающем** порядке (худший случай).

---

# **3. Промежуточные результаты сортировки**

Для массивов размерностью **n = 10**:

* Выводить **каждый шаг сортировки** (каждую итерацию внешнего цикла).
* На каждом шаге выводить **весь массив**.

---

# **4. Сравнение алгоритмов**

Для каждого алгоритма определить и вывести:

* количество **необходимых сравнений**;
* количество **перестановок (обменов)**.

Сравнение сделать по количеству этих операций.

---

# **5. Повторить сортировки для размеров:**

* **n = 100**
* **n = 1000**
* **n = 10000**

Для этих n **не выводить массив**, а показывать только:

* количество сравнений,
* количество перестановок.
